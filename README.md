# JCDSAH-025-Capstone-M2-Didiek-Arifman-
Capstone Project M2 mengenai materi SQL, data manipulation,  data visualization, dan statistics.

Dalam Capstone Project ini saya membuat rengkaian analisis dengan judul "Penjadwalan Dinamis Armada Transjakarta Berdasarkan Kepadatan Penumpang" menggunakan data Transjakarta Transaction, dimana didalam dataset tersebut terdiri dari 37900 baris data dan 22 kolom informasi mengenai ransID, payCardID, payCardBank, payCardName, payCardSex, payCardBirthDate, corridorID, corridorName, direction, tapInStops, tapInStopsName, tapinStopsLat, tapInStopsLon, stopStartSeq, tapInTime, tapOutStops, tapOutStopsName, tapOutStopsLat, tapOutStopsLon, stopEndSeq, tapOutTime, payAmount.

Setelah dilakukan pengecekan ditemukan Missing Value, Format data yang sesuai dan oulier untuk kemudian harus dilakukan data cleaning sehingga dataset tersebut nantinya dianalis.

Rangakaian Analisis yang dilakukan yaitu :
1. Analisis Pola Kepadatan Penumpang
2. Analisis Kesesuaian Feeder dengan Mainline Transjakarta
3. Analisis Kepadatan Koridor dan Halte

Hasil analisis yang didapatkan kemudian dapat diambil sebagai kesimpulan yang dapat digunakan dalam rekomendasi dan keputusan bisnin Transjakarta
